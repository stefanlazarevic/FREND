# Text

> This is the list of all usable classes for styling your text in paragraph, headings, spans etc.

```css
/* Align */

.text--left

.text--center

.text--right

.text--justify

/* Decoration */

.text--underline

.text--overline

.text--line-trough

/* Transformation */

.text--bold

.text--italic

.text--uppercase

.text--lowercase

.text--vertical

.text--capitalize

/* Creates three dots at the end of text. */
.text--overflow
```
